# 🐍 
A classic Snake Game built using **HTML, CSS, and JavaScript**.

Control the snake, collect apples, grow longer, and try to achieve the highest score possible without crashing into walls or yourself.

## 🎮 Features

* Smooth snake movement
* Score tracking
* Apple collection system
* Snake grows after eating apples
* Custom Game Over screen
* Play Again button
* Responsive and clean design
* Built without any external libraries

## 🕹️ How to Play

1. Open the game in your browser.
2. Use the arrow keys:

   * ⬆️ Up Arrow
   * ⬇️ Down Arrow
   * ⬅️ Left Arrow
   * ➡️ Right Arrow
3. Eat apples to increase your score.
4. Avoid hitting:

   * The walls
   * Your own snake body
5. If you crash, click **Play Again** to restart.

* HTML Canvas
