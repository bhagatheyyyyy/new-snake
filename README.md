🐍 My Snake Game

This is a simple Snake Game I made with HTML, CSS, and JavaScript.

You can play this game in your browser using the arrow keys.

I made this project to improve my JavaScript skills and learn how games work. through this, I learned about movement, score counting, collision detection, and using the HTML Canvas.

Features:
- Snake movement
- Score counter
- Game Over screen
- Restart button

Thank you for checking out my project 😊
